# Naming Conventions

Discover local style before recommending changes.

## Reference

Use `skillsets/shared/references/convention-discovery.md`.

## Check

- Nearby names for the same domain concepts.
- File, directory, type, function, variable, and test naming patterns.
- Acronym, casing, suffix, and prefix conventions.

## Recommend

- Preserve established domain vocabulary.
- Rename only when clarity or consistency improves.
- Avoid introducing parallel terms for existing concepts.
