# Language Conventions

Discover local style before recommending changes.

## Reference

Use `skillsets/shared/references/convention-discovery.md`.

## Check

- Existing language idioms, type usage, and standard-library choices.
- Formatter and linter configuration.
- Local preference for explicitness versus concision.

## Recommend

- Follow tool-enforced rules first.
- Match nearby idioms where tools are silent.
- Prefer clarity over clever syntax.
